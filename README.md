- Clone the project

```bash
  git clone https://github.com/mkdrx/adopt-me-app.git
```

- Go to the project directory

```bash
  cd adopt-me-app
```

- Install dependencies

```bash
  npm install
```

- Start the server

```bash
  npm start
```
